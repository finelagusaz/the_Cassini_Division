# ショートカットキー

t: ランダムトーク
m: メニューオープン
c: コミュニケート
d: コマンド
a: 同時起動中のゴーストとコミュニケート開始（対応ゴーストのみ）
g: 追加シェル「Gemini」用トーク

# Web拍手について

InputBoxからWeb拍手にメッセージを送ることが出来ます。
送信される内容はゴーストの名前、メッセージの2つで、送信者を特定する情報は一切、含まれていません。
